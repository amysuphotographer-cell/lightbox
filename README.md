# Image Gallery Viewer

一个优雅的图片评分与对比工具，采用莫兰迪配色风格，支持多文件夹对比、图片标注和智能报告生成。

## ✨ 功能特性

### 📁 图片加载
- 支持单个或多个文件夹加载
- 支持常见图片格式：JPG, PNG, GIF, WEBP, BMP
- 支持 RAW 格式：CR2, NEF, ARW, DNG 等
- 快速连续添加多个文件夹

### ⭐ 评分系统
- 三级评分：😍 喜欢、😐 一般、😢 不喜欢
- 支持为每张图片添加文字评论
- 多文件夹模式下可对同一图片的不同版本分别评分
- 实时统计显示评分分布和百分比

### 🔄 对比模式
- 点击图片名称查看同名图片在所有文件夹中的版本
- 每个版本独立缩放、拖动查看细节
- 并排展示各版本评分便于比较
- 清晰的文件夹标识

### ✏️ 画笔标记
- 点击✏️按钮进入标记模式
- 提供画笔、橡皮擦、颜色选择器
- 标记自动保存到浏览器本地
- 下次查看自动加载

### 📊 智能报告
- 自动归纳相似评论（基于Levenshtein距离算法）
- Goodcase：展示所有"喜欢"的评价和图片
- Badcase：展示"一般"和"不喜欢"的评价和图片
- 报告内容完全可编辑（标题、评论、图片文件名）
- 支持替换报告中的图片（鼠标悬停显示🔄按钮）
- 点击图片可放大查看，横向图片自动铺满屏幕

### 💾 导出功能
- 导出TXT格式文件分类清单（便于阅读）
- 导出JSON格式结构化数据（便于程序处理）
- 所有评分和标记自动持久化到浏览器本地存储

## 🚀 快速开始

### 在线使用
1. 下载 `index.html` 文件
2. 双击在浏览器中打开
3. 点击"选择文件夹"按钮开始使用

### 本地部署
```bash
# 克隆仓库
git clone https://github.com/your-username/image-gallery-viewer.git

# 进入目录
cd image-gallery-viewer

# 用浏览器打开
open index.html  # macOS
# 或
start index.html  # Windows
```

## 💡 使用技巧

1. **多文件夹对比**：适合对比同一组照片的不同处理版本（如不同修图软件的输出）
2. **画笔标记**：标记图片中需要关注的细节问题
3. **快速评分**：使用键盘快捷键快速为大量图片评分
4. **报告编辑**：生成报告后可直接编辑内容，适合制作演示文稿
5. **图片替换**：在报告中替换图片，方便制作最终版本

## 🔧 技术栈

- 纯前端实现，无需后端服务器
- HTML5 + CSS3 + Vanilla JavaScript
- Canvas API（用于画笔功能）
- LocalStorage（用于数据持久化）
- File System API（用于文件夹读取）
